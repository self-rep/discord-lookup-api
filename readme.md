# Discord Lookup API

## Prerequisites

- Discord Account
- Discord Bot Token

## How to use

---

```
 Step 1: Get a server
 Step 2: Install PHP
 Step 3: Drag your files into /var/www/html
 Step 4: http://<IP>/api.php?id=<DISCORD_ID>&output=<OUTPUT>
```

---

# Ouput Types

- JSON
- Plain Text
